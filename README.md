# Agrinho 🌱

**Agrinho** é um site educativo voltado para o ensino médio, focado em agricultura, sustentabilidade e tecnologia no campo. O projeto traz conteúdos, ferramentas e atividades interativas para estudantes aprenderem de forma prática e divertida.

## 💡 Funcionalidades

- **Conteúdos Educativos**: Artigos e dicas sobre agricultura e meio ambiente.  
- **Galeria de Plantas**: Informações e imagens de diferentes culturas.  
- **Calculadora Agrícola**: Ferramentas simples para plantio e colheita.  
- **Seção Interativa**: Quizzes e jogos para aprender sobre agricultura.  

## 🖥️ Tecnologias

- HTML5  
- CSS3  
- JavaScript  
- Bootstrap  

## 🎨 Layout e Tema

Visual moderno e limpo, com cores que remetem à natureza (verde, marrom, tons terrosos). Ícones de plantas e elementos agrícolas para navegação intuitiva e agradável.
